### Topics Covered
- Previous examples of microbiome meta-analyses
- 16S rRNA gene and shotgun metagenomic sequencing pipelines
- Reference databases for each sequencing platform
- Data sharing on SRA, ENA, and dbGaP
- Metadata sharing (e.g., NMDC) and IRB limitations

### Session Overview

This session covers meta-analysis approaches for microbiome studies and best practices for data sharing. We will discuss different sequencing platforms, reference databases, and data repository requirements.
