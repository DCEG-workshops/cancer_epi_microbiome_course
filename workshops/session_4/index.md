### Topics Covered
- Introduction to QIIME 2
- Bioinformatics pipeline for microbiome data
- Data processing and quality control
- Taxonomic assignment
- Diversity analysis

### Session Overview

This session introduces QIIME 2, the primary bioinformatics platform for microbiome data analysis. Led by Greg Caporaso, the primary author of QIIME 2, this virtual session will cover the complete pipeline from raw sequencing data to biological insights.

*Instructor: Greg Caporaso (Northern Arizona University)*
